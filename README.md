# Alumni-Association-Management-System
# 🚀 NexStep AI — Alumni & Student Networking Platform

A full-stack AI-powered SaaS platform that connects Students and Alumni for mentorship, referrals, and career growth.

---

# 🌟 Features

## 👨‍🎓 Student

* Find and connect with Alumni
* Send Mentorship Requests
* Request Job/Internship Referrals
* AI Resume Analyzer
* AI Career Roadmap Generator
* Smart Mentor Matching (based on skills)
* Real-time Chat

---

## 🎓 Alumni

* Accept/Reject Mentorship Requests
* Manage Referral Requests
* Chat with Students
* Donate to College Campaigns
* View Suggested Students

---

## 🏫 College Admin

* Manage Students & Alumni
* Create Donation Campaigns
* Monitor Reports & Activity
* View Top Alumni & Active Students

---

## 👑 Super Admin

* Manage Colleges
* Assign Admins
* Monitor Users
* Revenue Analytics Dashboard

---

# 🤖 AI Features

* Resume Analyzer (ATS Score + Suggestions)
* Career Roadmap Generator
* Smart Matching System

---

# 💬 Real-Time Features

* 1-to-1 Chat (Student ↔ Alumni)
* Group Discussions
* Online Status Indicator

---

# 💰 Monetization

* Students use token-based AI features
* Alumni usage is free

# ⚙️ Setup Instructions

## 1. Clone Repo

```bash
git clone https://github.com/ijharul/Campus-Nexus.git
cd Campus-Nexus
```

---

## 2. Install Dependencies

### Backend

```bash
cd server
npm install
```

### Frontend

```bash
cd client
npm install
```

---

## 3. Environment Variables

Create `.env` file in server:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
OPENAI_API_KEY=your_key
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret
```

---

## 4. Run Project

### Backend

```bash
npm run dev
```

### Frontend

```bash
cd client
npm run dev
```
